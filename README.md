# abc-count

Inicie este nuevo proyecto para un curso de NPM. Es acerca de escribir un texto y de forma automatica contara cuantos caracteres del abecedario hay en ese text.

## Install

```npm
npm install -g abc-count
```

# Usage

```bash
 abc-count
```
